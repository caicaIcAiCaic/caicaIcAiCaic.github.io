Pythagorean Triples

发表于 2024-04-24 更新于 2024-04-25 分类于 数论 Valine：
本文字数： 3.8k 阅读时长 ≈ 3 分钟

The Pythagorean Theorem, says that the sum of the squares of the sides of a right triangle equals the squares of the hypotenuse. In symbols,
$$a^{2}+b^{2}=c^{2}$$

There are many such triangles, The most famous has sides 3, 4, 5. Here are the first few examples:
$$3^{2}+4^{2}=5^{2}$$
$$5^{2}+12^{2}=13^{2}$$
$$8^{2}+15^{2}=17^{2}$$
$$28^{2}+45^{2}=53^{2}$$

Our first naive question is whether there are infinitely many Pythagorean triples. The anwser is absolutely ‘YES’.
If we take a Pythagorean triple $(a,b,c)$ and multiples it by some other number $d$, then we will obtain a new Pythagorean triple $(da, db, dc)$. This is because:
$$(da)^{2}+(db)^2=d^{2}(a^{2}+b^{2})=(dc)^{2}$$
But these new Pythagorean triples are not very interesting. So we will concentrate our attention on triples with no common factors, and we give them a name: A primitive Pythagorean triple. so, Is there are infinitely many primitive Pythagorean triples? To begin with, here are some primitive Pythagorean triples:
$$(3,4,5), (5,12,13), (8,15,17), (7,24,25), (20,21,29), (9,40,41)$$
A few conclusions can easily be drawn even from such a short list. For example, it looks like one of $a$ and $b$ is odd and the other even, and it also seems like that $c$ is always odd
.
It’s not hard to prove the conjecture: If (a, b, c) is a primitive Pythagorean triple, then one of a and b is odd and the other even, and c is odd.
First, if $a$ and $b$ are both even, then $c$ would also be even. that is
$$a=2x, b=2y, c=\sqrt{a^{2}+b^{2}}=2\sqrt{x^{2}+y^{2}}$$
so $a$, $b$ and $c$ have a common factor of 2, so the triple can not be primitive.
Next, suppose that $a$ and $b$ are both odd, then
$$a=2x+1, b=2y+1, c^{2}=2(2x^{2}+2x+2y^{2}+2y+1)$$
so $c^{2}$ is even, then $c$ must be even, because if we assume $c$ is odd $(c=2k+1)$, which means that $c^{2}=4(k^{2}+k)+1$, (can not even), so $c$ must be even
.
So we can suppose that $a=2x+1, y=2y+1, c=2z$,we can substitute these into the equation $a^{2}+b^{2}=c^{2}$ to get
$$2x^{2}+2x+2y^{2}+2y+1=2z^{2}$$
this equation says that an odd number is equal to an even number, which is impossible, so a and b can not both be odd.
if there is a primitive Pythagorean triple (a,b,c), we’ve just checked that they cannot both be even and odd, so it must be true that one is even and the other is odd, it’s then obvious that c is odd.

we can always switch a and b, so let assume that a is odd and b is even. Then we can factor
$$a^{2}=c^{2}-b^{2}=(c-b)(c+b)$$
If you list a few examples, you find that it looks like c-b and c+b are themselves always squares and they have no common factors.
$$3^{2}=5^{2}-4^{2}=(5-4)(5+4)=1*9$$
$$15^{2}=17^{2}-8^{2}=(17-8)(17+8)=9*25$$
$$35^{2}=37^{2}-12^{2}=(37-12)(37+12)=25*49$$
$$33^{2}=65^{2}-56^{2}=(65-56)(65+56)=9*121$$
We can prove the second assertion as follows. suppose that d is a common factor of c-b and c+b, so d divides $(c+b)+(c-b)=2c$ and $(c-b)-(c-b)=2b$, thus d divides 2b and 2c. But b and c have no common factors, so d must equal 1 or 2. But d also divides $(c-b)(c+b)=a^{2}$, and $a^{2}$ is odd, so d must be 1
.
We now know that c-b and c+b have no common factors, and their product is a square $a^{2}$. The only way that this can heppen is if c-b and c+b are themselves squares. So we can write
$$c+b=s^{2}, c-b=t^{2}, s>t>=1$$
s and t are both odd cause c is odd and b is even. besides, s and t also have no common factors. Then
$$c=\frac{s^{2}+t^{2}}{2}, b=\frac{s^{2}-t^{2}}{2}$$
and then
$$a=\sqrt{(c-b)(c+b)}=st$$
The following theorem records our accomplishment.
Pythagorean Triples Theorem: we will get every primitive Pythagorean Triple (a,b,c) with a odd b even by using the formulas
$$a=st,b=\frac{s^{2}-t^{2}}{2},c=\frac{s^{2}+t^{2}}{2}$$
where s>t>=1 are chosen to be any odd integers with no common factor.
But we only shown that if (a,b,c) is a PPT with a odd, then there are odd integers s>t>=1 with no common factors so that a, b, and c are given by these formulas. We still need to check that these formulas give a PPT.
It’s easy to get
$$(st)^{2}+(\frac{s^{2}-t^{2}}{2})^{2}=(\frac{s^{2}+t^{2}}{2})^2$$
we also need to check that $st,\frac{s^{2}-t^{2}}{2},\frac{s^{2}+t^{2}}{2}$ have no common factors. we postpone the poof in next blog.
